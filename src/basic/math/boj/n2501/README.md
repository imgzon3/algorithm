## 📃 문제 분석

- 수 n, k를 입력받고 n수의 k번째 약수를 출력한다.
- 단, n의 약수의 개수가 k보다 적을 경우, 0을 출력한다.

## 🗝️ 풀이 방법

- **구현**
  - 반복문을 활용하여 약수를 계산한다.
  - 약수의 총 개수보다 k가 클 경우, 0을 출력해야 함
