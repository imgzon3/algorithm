## 📃 문제 분석

- 수를 입력받고, 해당 수의 완전수 여부에 따라 다른 결과를 출력하시오.

## 🗝️ 풀이 방법

- **구현**
  - 입력받은 수의 약수를 반복문을 활용하여 모두 구하고, ```ArrayList```에 저장
  - 위 리스트의 값을 모두 더한 값이 첫 번째 입력 값과 같다면, 문제 조건대로 출력한다.
  - 아니라면, 마찬가지로 문제 조건대로 출력한다.
