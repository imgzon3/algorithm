## 📃 문제 분석

- 입력받은 숫자를 소인수분해한 결과를 출력

## 🗝️ 풀이 방법

- **구현**
  - 입력받은 숫자가 1이 될때까지 ```while(true)``` 반복을 한다.
    - 2를 나누어본다.
    - 만일 나누어지는 경우, 출력 한번
    - 안나누어지는 경우, 나누는 수를 +1
  - 여기서 핵심은, 소수만 나눌 생각보다는 2, 3, 4, 5, ... 이렇게 나누어도 상관없다는 것!
  - 어짜피 해당 숫자보다 작은 소수로 이미 다 나누어진 상태이기 때문이다.
  - 1이면 아무것도 출력하지 않음 유의하기
