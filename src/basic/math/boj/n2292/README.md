## 📃 문제 분석

- 수를 입력받고, 해당 방까지 몇개의 방을 지나가는지 계산하기

## 🗝️ 풀이 방법

- **구현**
  - 규칙을 확인해보자
    - 1
    - 2 ~ 7 (6개)
    - 8 ~ 19 (12개)
    - 20 ~ 37 (18개)
  - 첫 번째 방을 제외하면, 중앙의 방에서 멀어질수록 숫자가 6, 12, 18, ... 순으로 커진다.
  - 해당 규칙을 활용하여 문제를 해결하자
