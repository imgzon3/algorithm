## 📃 문제 분석

- 문장 입력받고, 단어 개수 구하기

## 🗝️ 풀이 방법

- **구현**
  - ```StringTokenizer```로 문자열을 받고, ```countTokens()```메서드를 활용해 단어의 개수를 구한다.

## 💻 핵심 코드

```java
public class Main {
    public static void main(String[] args) {
       System.out.println(st.countTokens());
    }
}
```
